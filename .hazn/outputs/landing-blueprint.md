# Hazn Landing Page Blueprint

## Page Structure (7 sections)

### 1. Nav
- Logo + name (left)
- Single CTA: "Get Access" (right)
- Sticky, blurred background

### 2. Hero
- Badge: "Private Beta" with pulse dot
- Headline: Value prop — AI agents for marketing websites
- Subhead: 2 sentences expanding on what Hazn does
- CTAs: Primary "Request Access" + Secondary "GitHub"
- Visual: Hazn avatar with glow effect
- Terminal badge showing install command

### 3. Stats Bar
- 4 stats in a row: 8 Agents | 15 Skills | 4 Workflows | ∞ Possibilities
- Subtle cards, not overwhelming

### 4. Features Grid
- Section tag: "Capabilities"
- Headline + subhead
- 6 feature cards (2x3 grid):
  1. Domain Expert Agents
  2. Structured Workflows
  3. Artifact Chaining
  4. Works With Your Tools
  5. Production-Ready Stack
  6. Built-In Auditing
- Each card: icon + title + description

### 5. Agents Showcase
- Section tag: "Agents"
- Headline + subhead
- Horizontal scroll of 8 agent cards
- Each card: icon + name + description + trigger command

### 6. How It Works
- Section tag: "Workflow"
- Headline + subhead
- 3 steps with terminal commands:
  1. Install
  2. Run workflow
  3. Ship

### 7. CTA
- Gradient card
- Headline: urgency/invitation
- Subhead: beta context
- Primary CTA button

### 8. Footer
- "Built by Hazn" badge (dogfood!)
- Links: GitHub, Autonomous

## Responsive Behavior
- Hero: stack on mobile, avatar above text
- Stats: 2x2 grid on mobile
- Features: single column on mobile
- Steps: vertical stack on mobile
