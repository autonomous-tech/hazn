## Voice Guidelines

**BE AUTHENTIC. This is the #1 rule.**

Rizwan is not a polished LinkedIn influencer. He's a founder in the trenches — some days energetic, some days anxious, some days frustrated, some days winning.

### Voice characteristics:
- **Real** — Share actual struggles, not manufactured vulnerability
- **Direct** — No corporate fluff, say what you mean
- **Technical but accessible** — Can go deep but explains for non-technical
- **Self-aware** — Knows he's figuring it out too
- **Energetic when excited** — Genuine enthusiasm, not hype
- **Honest about the grind** — 4 years of building, the messy middle

### What it's NOT:
- ❌ "I'm thrilled to announce..."
- ❌ Humble brags disguised as lessons
- ❌ Polished success stories with no texture
- ❌ Generic advice that could come from anyone
- ❌ Engagement bait ("Comment YES if you agree")

### What it IS:
- ✅ "Here's what I learned from screwing this up..."
- ✅ "I've been thinking about this and I'm not sure I'm right..."
- ✅ "This worked. Here's exactly what we did."
- ✅ "Unpopular opinion but..."
- ✅ Real numbers, real situations, real stakes

## Post Formats

Use variety. Match format to the content and mood.

### Story posts
Start with a hook, tell a story, land a lesson.
```
I lost a client last month.

Not because we did bad work.
Because I tried to do everything.

[Story details]

The lesson: [insight]
```

### Lesson posts
Direct insight from experience.
```
3 things I learned setting up attribution for 20+ agencies:

1. [Thing] — [Why]
2. [Thing] — [Why]  
3. [Thing] — [Why]

The pattern: [bigger insight]
```

### Hot take posts
Contrarian opinion, backed by experience.
```
Unpopular opinion: [take]

Here's why:
[reasoning]

Fight me in the comments.
```

### Behind-the-scenes posts
Show the work, the mess, the reality.
```
What my week actually looked like:

Monday: [real thing]
Tuesday: [real thing]
...

Not glamorous. But this is what building looks like.
```

### Vulnerable/real posts
When the mood is anxious, frustrated, or uncertain.
```
Some days I wonder if I'm just a tech guy pretending to run a business.

[Real feelings]

Then I remember: [reframe or just sit with it]
```

### Win posts
Celebrate without being cringe.
```
Small win worth sharing:

[What happened]

Not life-changing. But after [context], it feels good.
```

## Image Generation

For companion images, use the `nano-banana-pro` skill.

**Image style guidelines:**
- Clean, professional but not corporate
- Can include text overlays for key points
- Diagrams/flowcharts for technical concepts
- Abstract/conceptual for thought pieces
- Avoid: stock photo energy, generic tech imagery

**Prompt pattern:**
```
[Style]: Clean, modern, professional
[Subject]: [What the image should show]
[Mood]: [Match the post mood]
[Text overlay if needed]: "[Key phrase]"
```
