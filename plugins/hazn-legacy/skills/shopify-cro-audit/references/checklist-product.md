# Product Page CRO Checklist

## Product Images

- [ ] **Multiple images** — Minimum 4-6 angles/views
- [ ] **Zoom functionality** — Hover or tap to zoom works
- [ ] **Lifestyle shots** — Product in use, not just white background
- [ ] **Video if applicable** — Demo or 360° view
- [ ] **Image gallery UX** — Thumbnails visible, swipe works on mobile
- [ ] **Consistent sizing** — All images same aspect ratio

## Price & Buy Box

- [ ] **Price prominent** — Large, clear font, above fold
- [ ] **Compare-at price** — If on sale, original price struck through
- [ ] **Savings shown** — "Save $20" or "20% off" visible
- [ ] **Buy button above fold** — Visible without scrolling on desktop
- [ ] **Buy button contrast** — High contrast color, minimum 44px height
- [ ] **Sticky buy button** — Follows scroll on mobile
- [ ] **Express checkout** — Shop Pay, Apple Pay, PayPal visible

## Product Information

- [ ] **Title clear** — Descriptive, includes key attribute (size, color, model)
- [ ] **Description scannable** — Bullets, short paragraphs, not walls of text
- [ ] **Benefits over features** — "Stay warm all winter" not just "100% wool"
- [ ] **Size/variant selector** — Easy to use, shows availability
- [ ] **Size guide** — Link or modal for sizing help
- [ ] **Shipping info** — Delivery estimate visible before add to cart
- [ ] **Return policy** — Clear, linked or expandable

## Social Proof

- [ ] **Star rating visible** — Above fold, near title
- [ ] **Review count** — "Based on 127 reviews"
- [ ] **Review snippets** — 2-3 highlighted reviews visible
- [ ] **Photo reviews** — Customer photos in review section
- [ ] **Review filtering** — By rating, recency, verified purchase
- [ ] **Q&A section** — If applicable for complex products

## Urgency & Scarcity

- [ ] **Stock indicator** — "Only 3 left" when low
- [ ] **Recently purchased** — "Sarah in NYC bought this 2h ago"
- [ ] **Limited time offer** — Countdown if promotion active
- [ ] **Seasonal urgency** — "Order by Dec 20 for Christmas delivery"

## Cross-Sells & Upsells

- [ ] **Frequently bought together** — Bundle suggestion
- [ ] **You may also like** — Related products visible
- [ ] **Recently viewed** — Easy to compare alternatives
- [ ] **Upsell variant** — "Upgrade to premium for $10 more"

## Trust Elements

- [ ] **Money-back guarantee** — Visible near buy button
- [ ] **Secure checkout badge** — Lock icon or trust seal
- [ ] **Brand story** — Brief "Why us" if not well-known brand
- [ ] **Warranty info** — If applicable

## Mobile Specific

- [ ] **Image swipe** — Smooth gallery navigation
- [ ] **Sticky CTA** — Add to cart always accessible
- [ ] **Collapsible sections** — Description, reviews, shipping in accordions
- [ ] **Quick add from gallery** — Don't require scrolling down

## Scoring Guide

| Score | Criteria |
|-------|----------|
| 9-10 | Conversion-optimized, exceeds best practices |
| 7-8 | Strong product page, minor optimizations needed |
| 5-6 | Average Shopify product page, clear gaps |
| 3-4 | Missing key conversion elements |
| 1-2 | Broken or severely underoptimized |
