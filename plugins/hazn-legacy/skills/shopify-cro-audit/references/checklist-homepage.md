# Homepage CRO Checklist

## Above the Fold (Hero Section)

- [ ] **Value proposition visible** — Can visitors understand what you sell in <3 seconds?
- [ ] **Primary CTA present** — Shop Now / Browse Collection / Get Started clearly visible
- [ ] **CTA contrast** — Button stands out from background (AA contrast minimum)
- [ ] **Hero image quality** — High-res, properly cropped for desktop and mobile
- [ ] **No slider/carousel** — Static hero converts 25% better than rotating banners
- [ ] **Mobile hero works** — Text readable, CTA thumb-accessible

## Navigation

- [ ] **Logo links home** — Standard convention users expect
- [ ] **Categories clear** — Max 7±2 top-level items (Miller's Law)
- [ ] **Search prominent** — Visible icon or field, not hidden in hamburger
- [ ] **Cart icon with count** — Badge showing item count
- [ ] **Mega menu if needed** — For stores with 50+ products
- [ ] **Mobile hamburger works** — Smooth animation, all links accessible

## Trust & Social Proof

- [ ] **Review aggregate** — "Rated 4.8★ by 2,500+ customers"
- [ ] **Press logos** — "As seen in..." if applicable
- [ ] **Security badges** — SSL, payment icons in footer
- [ ] **Money-back guarantee** — Prominently displayed
- [ ] **Real customer photos** — UGC if available

## Product Display

- [ ] **Featured products** — Best sellers or new arrivals visible
- [ ] **Quick add to cart** — Don't force users to product page for simple items
- [ ] **Price visible** — No "click for price" patterns
- [ ] **Sale prices clear** — Compare-at price crossed out
- [ ] **Collection links** — Easy navigation to browse categories

## Performance

- [ ] **First Contentful Paint** — < 1.8s (Good), < 3s (Needs Improvement)
- [ ] **Largest Contentful Paint** — < 2.5s (Good)
- [ ] **Cumulative Layout Shift** — < 0.1 (Good)
- [ ] **Images optimized** — WebP format, lazy loading enabled
- [ ] **No render-blocking resources** — JS/CSS properly deferred

## Mobile Specific

- [ ] **Touch targets** — Minimum 44x44px for tappable elements
- [ ] **No horizontal scroll** — Content fits viewport
- [ ] **Sticky header** — Compact header on scroll with cart access
- [ ] **Bottom nav optional** — For stores with 4-5 key actions
- [ ] **Fast mobile load** — < 3s on 4G connection

## Scoring Guide

| Score | Criteria |
|-------|----------|
| 9-10 | All critical elements present, excellent execution |
| 7-8 | Minor issues, above average for Shopify stores |
| 5-6 | Average, noticeable improvement opportunities |
| 3-4 | Below average, significant conversion leaks |
| 1-2 | Critical issues, immediate attention needed |
