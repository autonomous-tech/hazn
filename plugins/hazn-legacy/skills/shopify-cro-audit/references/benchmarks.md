## Speed & Performance Impact

Source: Google, WebFX, Think with Google

| Improvement | CVR Impact |
|-------------|------------|
| 1 second faster load time | +7% CVR |
| Load time under 2 seconds | +15% CVR |
| Each additional second on mobile | -20% CVR |
| Negative mobile experience | -62% future purchases |

### Core Web Vitals Targets
| Metric | Good | Needs Improvement | Poor |
|--------|------|-------------------|------|
| LCP (Largest Contentful Paint) | ≤2.5s | 2.5-4s | >4s |
| FID (First Input Delay) | ≤100ms | 100-300ms | >300ms |
| CLS (Cumulative Layout Shift) | ≤0.1 | 0.1-0.25 | >0.25 |
| FCP (First Contentful Paint) | ≤1.8s | 1.8-3s | >3s |
| TTFB (Time to First Byte) | ≤800ms | 800-1800ms | >1800ms |

## Calculation Formulas

### Conversion Rate
```
CVR = (Conversions / Total Visitors) × 100
```

### Revenue Impact of CVR Improvement
```
Additional Revenue = Monthly Visitors × CVR Increase × AOV

Example:
- 100,000 monthly visitors
- CVR increase from 1.5% to 2.0% (+0.5%)
- AOV = $75

Additional Revenue = 100,000 × 0.005 × $75 = $37,500/month
```

### Cart Abandonment Rate
```
Cart Abandonment = (Carts Created - Purchases) / Carts Created × 100
```

### Add-to-Cart Rate
```
ATC Rate = Add-to-Cart Actions / Product Page Views × 100
```
