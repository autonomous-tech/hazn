# Cart & Checkout CRO Checklist

## Cart Page

### Cart Summary
- [ ] **Product images visible** — Thumbnail for each item
- [ ] **Easy quantity edit** — +/- buttons or input field
- [ ] **Remove item** — Clear X or "Remove" link
- [ ] **Price per item** — Unit price visible
- [ ] **Line item total** — Quantity × price shown
- [ ] **Subtotal clear** — Before shipping/tax

### Checkout CTAs
- [ ] **Primary checkout button** — High contrast, above fold
- [ ] **Secondary checkout** — Bottom of cart for long carts
- [ ] **Express checkout** — Shop Pay / Apple Pay / PayPal Express
- [ ] **Continue shopping** — Link back to store (secondary styling)

### Trust & Assurance
- [ ] **Security badges** — Payment icons, SSL indicator
- [ ] **Return policy reminder** — "Free 30-day returns"
- [ ] **Support access** — Chat or help link visible

### Upsells & Offers
- [ ] **Free shipping progress** — "Add $15 more for free shipping"
- [ ] **Cart upsell** — Relevant product recommendation
- [ ] **Gift wrapping option** — If applicable
- [ ] **Promo code field** — Visible but not dominant

### Mobile Cart
- [ ] **Sticky checkout button** — Always visible
- [ ] **Slide-out cart** — Ajax cart instead of page redirect
- [ ] **Easy editing** — Quantity/remove works smoothly

---

## Checkout Page (Shopify Checkout)

### Guest Checkout
- [ ] **No login required** — Guest checkout prominent option
- [ ] **Express checkout first** — Shop Pay / GPay / Apple Pay at top
- [ ] **Social login** — If login needed, offer Google/Facebook

### Form Optimization
- [ ] **Minimal fields** — Only essential info required
- [ ] **Autofill works** — Browser autofill properly configured
- [ ] **Address autocomplete** — Google Places or similar
- [ ] **Phone format flexible** — Accepts various formats
- [ ] **Error messages clear** — Inline validation, specific errors
- [ ] **Required fields marked** — Asterisk or "(required)" label

### Progress Indication
- [ ] **Steps visible** — Information → Shipping → Payment
- [ ] **Current step highlighted** — Clear visual indicator
- [ ] **Step count** — "Step 2 of 3"

### Order Summary
- [ ] **Always visible** — Sticky sidebar on desktop
- [ ] **Expandable on mobile** — Collapsible summary
- [ ] **Product thumbnails** — Visual confirmation
- [ ] **Editable** — Link back to cart
- [ ] **Promo code visible** — Easy to apply discount

### Payment
- [ ] **Multiple methods** — Credit, PayPal, Shop Pay minimum
- [ ] **Local payments** — Klarna, Afterpay for target markets
- [ ] **Payment icons** — Visa, MC, Amex logos visible
- [ ] **Secure indicator** — Lock icon, "Secure checkout"
- [ ] **Billing = shipping default** — Checkbox pre-checked

### Shipping
- [ ] **Delivery estimates** — "Arrives Dec 15-17"
- [ ] **Multiple options** — Standard, express, overnight
- [ ] **Price visible** — No hidden shipping surprises
- [ ] **Free shipping threshold** — "Free shipping on orders $50+"

### Final Review
- [ ] **Order total clear** — Subtotal + shipping + tax = total
- [ ] **Place order button** — High contrast, clear copy
- [ ] **Terms/privacy links** — Accessible but not intrusive

---

## Post-Purchase

- [ ] **Order confirmation page** — Clear success message
- [ ] **Order number visible** — Easy to copy/reference
- [ ] **Email confirmation** — Immediate send
- [ ] **Tracking info** — When available, prominent display
- [ ] **Cross-sell opportunity** — "Customers also bought" on thank you page
- [ ] **Account creation** — Optional, after purchase

---

## Cart Abandonment Prevention

- [ ] **Exit intent popup** — Offer or reminder on cart exit
- [ ] **Abandoned cart emails** — Klaviyo/Omnisend configured
- [ ] **Persistent cart** — Cart survives browser close
- [ ] **Cart recovery SMS** — If phone collected

## Scoring Guide

| Score | Criteria |
|-------|----------|
| 9-10 | Frictionless checkout, optimized for conversion |
| 7-8 | Good checkout flow, minor friction points |
| 5-6 | Standard Shopify checkout, typical issues |
| 3-4 | Significant friction, cart abandonment likely |
| 1-2 | Broken checkout, major blockers |
