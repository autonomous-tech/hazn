Accessibility Statement

International Democracy Union — idu.org

Our Commitment

The International Democracy Union is committed to ensuring this website is accessible to the widest possible audience, including people with disabilities. We aim to conform to the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA. We continually review our website to improve its accessibility.

Measures Taken

The IDU has taken the following measures to ensure accessibility of this website:

  • — Semantic HTML structure, using appropriate heading levels and landmark regions
  • — Keyboard navigation support throughout all interactive elements
  • — Sufficient colour contrast ratios meeting WCAG 2.1 AA requirements
  • — Descriptive alt text on all meaningful images
  • — Responsive design ensuring full functionality across all screen sizes and devices

Known Issues

We are actively working to improve the accessibility of this website. Some content, particularly older documents in PDF format, may not yet fully meet accessibility standards. If you encounter barriers while using this site, please contact us so we can address the issue.

Feedback

We welcome feedback on the accessibility of this website. If you experience any barriers or have suggestions for improvement, please contact us:

accessibility@idu.org

We aim to respond to accessibility enquiries within 5 working days.
