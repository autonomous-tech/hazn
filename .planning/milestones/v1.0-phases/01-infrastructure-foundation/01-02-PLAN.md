core app models (Agency, EndClient, VaultCredential):
- Agency: UUID PK, name, slug (unique), house_style (JSON), methodology (JSON), tool_preferences (JSON), created_at, updated_at
- EndClient: UUID PK, agency FK (CASCADE), name, slug, competitors (JSON list), created_at, updated_at. unique_together: [agency, slug]
- VaultCredential: UUID PK, agency FK (CASCADE, nullable), end_client FK (CASCADE, nullable), service_name (max 100), vault_secret_id (max 500, VARCHAR -- NOT the secret itself), created_at, updated_at. unique_together: [end_client, service_name]

marketing app models (Keyword, Audit, Campaign, Decision):
- Keyword: UUID PK, end_client FK (CASCADE), term (max 500), search_volume (int, nullable), difficulty (float, nullable), intent (max 50), status (max 50, default "discovered"), metadata (JSON), created_at, updated_at
- Audit: UUID PK, end_client FK (CASCADE), audit_type (max 100), findings (JSON), score (float, nullable), created_at
- Campaign: UUID PK, end_client FK (CASCADE), name (max 255), campaign_type (max 100), status (max 50, default "draft"), config (JSON), created_at, updated_at
- Decision: UUID PK, end_client FK (CASCADE), campaign FK (SET_NULL, nullable), decision_type (max 100), rationale (text), outcome (JSON), created_at

content app models (BrandVoice, ApprovedCopy):
- BrandVoice: UUID PK, end_client FK (CASCADE), content (text), embedding (VectorField 1536, nullable), version (positive int, default 1), is_active (bool, default True), created_at. UniqueConstraint: [end_client, is_active] where is_active=True (name: unique_active_brand_voice_per_client)
- ApprovedCopy: UUID PK, end_client FK (CASCADE), copy_type (max 100), content (text), embedding (VectorField 1536, nullable), campaign FK to marketing.Campaign (SET_NULL, nullable), version (positive int, default 1), is_active (bool, default True), created_at
