Create analytics_server.py with Django setup guard and FastMCP("hazn-analytics"):

1. Private helpers:
   - _get_ga4_client(l2_agency_id, l3_client_id): queries VaultCredential.objects.get(end_client_id=l3_client_id, service_name="ga4"), calls read_secret to get service account JSON, creates Credentials.from_service_account_info with analytics.readonly scope, returns BetaAnalyticsDataClient(credentials=credentials)
   - _get_gsc_service(l2_agency_id, l3_client_id): same pattern with service_name="gsc" and webmasters.readonly scope, returns build("searchconsole", "v1", credentials=credentials)

2. @mcp.tool() pull_ga4_data:
   - Creates client via _get_ga4_client
   - Runs a RunReportRequest with DateRange(start_date=f"{days}daysAgo", end_date="today"), dimensions=["date"], metrics=["sessions", "totalUsers", "screenPageViews", "bounceRate"]
   - Runs a second report for top pages: dimensions=["pagePath"], metrics=["sessions", "totalUsers"], order by sessions desc, limit=10
   - Runs a third report for top sources: dimensions=["sessionSource"], metrics=["sessions"], order by sessions desc, limit=10
   - Returns aggregated dict

3. @mcp.tool() query_gsc:
   - Creates service via _get_gsc_service
   - Calls service.searchanalytics().query(siteUrl=site_url, body={startDate, endDate, dimensions: ["query"], rowLimit: limit}).execute()
   - Also calls with dimensions: ["page"] for top pages
   - Returns structured dict

4. Error handling: wrap all Google API calls in try/except (Exception), return {"error": str(e)}
