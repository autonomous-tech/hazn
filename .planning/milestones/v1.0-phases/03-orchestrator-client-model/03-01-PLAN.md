  <action>
    Create Pydantic models for workflow YAML validation and a parser module:

    1. Create `workflow_models.py` with Pydantic v2 models:

    **WorkflowPhaseSchema(BaseModel):**
    - id: str
    - name: str
    - agent: str | None = None
    - depends_on: list[str] = Field(default_factory=list)
    - outputs: list[str] = Field(default_factory=list)
    - required: bool = True
    - tools: list[str] = Field(default_factory=list)
    - skills: list[str] = Field(default_factory=list)
    - actions: list[str] = Field(default_factory=list)
    - command: str | None = None
    - note: str | None = None
    - optional_inputs: list[str] = Field(default_factory=list)
    - parallel_tracks: list[str] = Field(default_factory=list)
    - extra_config: dict = Field(default_factory=dict)  # catches per_article, article_template, etc.
    - model_config = ConfigDict(extra="allow")  # allows unknown YAML keys to pass through

    **WorkflowCheckpoint(BaseModel):**
    - after: str
    - message: str

    **WorkflowSchema(BaseModel):**
    - name: str
    - description: str
    - trigger: str
    - phases: list[WorkflowPhaseSchema]
    - checkpoints: list[WorkflowCheckpoint] = Field(default_factory=list)
    - estimated_duration: dict | str | None = None
    - deliverables: list[str] = Field(default_factory=list)
    - model_config = ConfigDict(extra="allow")  # allows article_template, etc.

    2. Create `workflow_parser.py` with functions:

    **load_workflow(path: str | Path) -> WorkflowSchema:**
    - Open file, yaml.safe_load, WorkflowSchema.model_validate(data)
    - Raises FileNotFoundError if path missing, pydantic.ValidationError if schema invalid

    **get_dependency_graph(workflow: WorkflowSchema) -> dict[str, set[str]]:**
    - Build dict of phase_id -> set(depends_on)
    - For phases with parallel_tracks: create sub-phases with no inter-dependency

    **get_execution_order(workflow: WorkflowSchema) -> list[set[str]]:**
    - Use graphlib.TopologicalSorter to compute execution waves
    - Returns list of sets where each set is phases that can run in parallel
    - Raises graphlib.CycleError if circular dependencies detected

    All 7 existing YAML files must parse without error. The parser must be tolerant of schema variations (per_article, parallel_tracks, article_template at root) via Pydantic's extra="allow" config.
