## Architecture

```
User Request
     ↓
  Workflow (orchestrates phases)
     ↓
  Agent (guides conversation, applies expertise)
     ↓
  Skills (detailed domain knowledge)
     ↓
  Outputs (artifacts in .hazn/outputs/)
```
