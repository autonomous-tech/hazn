Frequently Asked Questions

What is Hazn?

Hazn is an AI-driven marketing website development framework. It transforms AI coding assistants like Claude Code, Cursor, and Windsurf into a team of 12 specialized marketing experts. With 25 domain skills, 7 structured workflows, and a full GA4/GSC analytics audit pipeline, Hazn enables developers to build high-converting B2B websites and deliver professional analytics reports ‚Äî all with expert-level strategy, copy, and SEO built in.

How is Hazn different from other AI website builders?

Traditional AI tools generate code. Hazn generates strategy-informed, conversion-optimized websites. Each phase builds on the last through artifact chaining: Strategy informs UX, UX informs copy, copy informs code. The result is websites designed to convert‚Äînot just websites that exist.

What AI agents are included in Hazn?

Hazn includes 12 expert AI agents:

  • Strategist ‚Äî Market positioning, audience definition, competitive analysis
  • UX Architect ‚Äî Page blueprints, information architecture, user flows
  • Copywriter ‚Äî Headlines, value props, CTAs, conversion copy
  • Wireframer ‚Äî Mid-fidelity layouts for stakeholder validation
  • Developer ‚Äî Next.js + Payload CMS + Tailwind implementation
  • SEO Specialist ‚Äî Technical SEO, schema, content optimization
  • Content Writer ‚Äî SEO + AEO + GEO optimized blog posts
  • Auditor ‚Äî Conversion, copy, visual, and SEO audits
  • Analytics Audit Pipeline ‚Äî Full GA4/GSC MarTech & attribution audit with adversarial review and branded HTML client report (4 specialist sub-agents)

Is Hazn free to use?

Yes, Hazn is 100% open source and free under the MIT license. It's available on GitHub at github.com/autonomous-tech/hazn. You only need an AI coding assistant subscription (Claude Code, Cursor, or Windsurf) to use it.

What coding assistants work with Hazn?

Hazn works with Claude Code, Cursor, Windsurf, and Clawdbot. Install once, use everywhere. The skills and agents are designed to work with any AI coding assistant that supports MCP (Model Context Protocol) or custom instructions.

Can Hazn help with AI search optimization?

Yes, Hazn includes dedicated AI SEO skills for GEO (Generative Engine Optimization) and AEO (Answer Engine Optimization). The SEO Specialist and AI SEO Optimizer skills help your content get cited by ChatGPT, Claude, Perplexity, and other AI systems.

What tech stack does Hazn use?

Hazn's Developer agent builds websites using Next.js 14, Payload CMS, and Tailwind CSS. This modern stack follows best practices for performance, SEO, and maintainability. The framework produces production-ready code.
